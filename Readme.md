## Overview

This project is an interactive and fully-featured implementation of **Conway’s Game of Life**, developed using **C++** and the **SFML** library for graphics and user interaction. It simulates cellular automata behavior with real-time updates and visualization.

Conway’s Game of Life is a zero-player game where the system evolves based on its initial state, following a set of rules for cell survival, death, and reproduction. Despite its simplicity, it can exhibit incredibly complex behavior, making it a fascinating tool for exploring emergent systems.

---

## What You Can Do with This Project

### 🔄 Run and Explore Cellular Automata
- Watch how simple patterns evolve over time.
- Experiment with well-known structures like **gliders**, **oscillators**, and **spaceships**.
- Discover your own patterns by placing cells freely.

### ✍️ Edit and Design Custom Boards
- Pause the simulation at any time.
- Add or remove live cells directly through the interface.

### 💾 Save and Load Your Simulations
- Save your current state to a file for later review.
- Load previously saved simulations and continue evolving them.

---

## Technical Highlights

- Built using **SFML (Simple and Fast Multimedia Library)** for smooth graphics and input handling.
- Modular and object-oriented architecture for easy extension.
- Real-time rendering and responsive controls.
- Logging system with customizable output.
- File I/O support for saving/loading board states.
